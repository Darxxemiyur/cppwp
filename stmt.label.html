8 Statements [stmt.stmt]

8.2 Labeled statement [stmt.label]

1
#
A statement can be labeled.
labeled-statement:
attribute-specifier-seqopt identifier : statement
attribute-specifier-seqopt case constant-expression : statement
attribute-specifier-seqopt default : statement
The optional attribute-specifier-seq appertains to the label.
The only use of a label with an identifier is as the target of a goto.
No two labels in a function shall have the same identifier.
A label can be used in a goto statement before its introduction by a labeled-statement.
2
#
Case labels and default labels shall occur only in switch statements.
