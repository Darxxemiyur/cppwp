This subclause contains templates that may be used to query properties of types at compile time.

struct extent;
If T is not an array type, or if it has rank less than or equal to I, or if I is 0 and T has type “array of unknown bound of U”, then 0; otherwise, the bound ([dcl.array]) of the Ith dimension of T, where indexing of I is zero-based
